# Arma Ballistic Calculator

Баллистический калькулятор-ассистент для **Arma Reforger**.

Считывает данные с экрана (дальность, ветер, азимут) и рассчитывает поправки на прицел в милрадах.

## Как работает

1. **Зажимаешь R** на 3 секунды (дальномер активируется)
2. **Отпускаешь R** — ждём 2 сек (данные стабилизировались)
3. **OCR считывает** с экрана: дальность, ветер, азимут
4. **Расчёт** баллистической поправки (Elevation + Windage)
5. **Overlay** показывает результат в левом верхнем углу на 3 секунды

## Установка

```bash
pip install -r requirements.txt
```

### Зависимости

- **Tesseract OCR** — нужен для распознавания текста
  - [Windows](https://github.com/UB-Mannheim/tesseract/wiki)
  - `pip install pytesseract` + указать путь к tesseract.exe в PATH

## Запуск

```bash
python main.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| **R (зажать 3с)** | Запуск расчёта |
| **1** | 7.62x51 NATO M80 Ball |
| **2** | 7.62x51 NATO M993 AP (по умолчанию) |
| **3** | 5.56x45 NATO M855 |
| **4** | .338 Lapua Magnum |
| **5** | .50 BMG M33 Ball |
| **Q** | Выход |

## Вывод

Overlay отображает:
- **↑ Elevation** — поправка на дальность (мрады)
- **→ Windage** — поправка на ветер (мрады)

## Структура

```
arma-ballistic-calc/
├── main.py              # Точка входа, машина состояний
├── requirements.txt     # Зависимости Python
├── src/
│   ├── ballistic_calc.py  # Баллистический калькулятор
│   ├── ocr_reader.py      # OCR модуль (дальность, ветер, азимут)
│   └── overlay.py         # Pygame overlay (always-on-top)
├── configs/             # Конфигурации оружия/калибров
└── assets/              # Скриншоты для калибровки OCR
```

## Примечание

Это **не чит** — утилита считывает визуальную информацию с экрана и рассчитывает поправки по формулам. Не взаимодействует с памятью игры, не модифицирует файлы, не инжектирует код.
